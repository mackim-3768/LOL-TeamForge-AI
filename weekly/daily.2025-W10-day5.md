### 3. AI 분석 점수 세부 항목 분해 (Detailed AI Score Breakdown)
- **적용 위치**: AI 분석 결과 모달 또는 섹션

- **가치**: 라인전, 생존력, 시야 장악 등으로 점수를 세분화하여 설명 가능성(Explainability)을 강화한다.

- **가치**: 추천의 현실성과 실용성을 높인다. (Riot API의 Mastery 정보 활용)

### 5. 분석 요약 클립보드 복사 기능 (Copy Analysis Summary to Clipboard)
- **적용 위치**: 소환사 상세 및 AI 분석 페이지
- **해결 문제**: 친구나 커뮤니티에 자신의 분석 결과를 공유하고 싶을 때 스크린샷을 찍거나 일일이 타이핑해야 한다.
- **가치**: 별도의 소셜 기능 없이도 사용자 간의 자연스러운 바이럴을 유도할 수 있는 가성비 좋은 기능이다.

## 2. Triage

### Selected: 최근 전적 성과 추이 시각화 (Match Performance Trend Visualization)

**선정 이유**:
- **Impact (High)**: 텍스트 위주의 현재 UI에 강력한 시각적 요소를 더해 서비스의 완성도를 크게 높여준다.
- **Confidence (High)**: `recharts`나 `chart.js` 같은 성숙한 라이브러리를 활용하면 구현 리스크가 낮다.
- **Ease (Medium)**: 백엔드 추가 작업 없이 프론트엔드 작업 위주로 진행 가능하여 빠른 이터레이션이 가능하다.
- **Portfolio Value**: 데이터 시각화(Data Visualization) 역량을 보여주기에 가장 적합한 기능이다.

**다른 아이디어 보류 사유**:
- **티어 평균 비교**: 비교를 위한 기준 데이터(Reference Data)를 먼저 확보하거나 하드코딩해야 하는 선결 과제가 있다.
- **AI 점수 세부 항목**: 백엔드 및 AI 모델(또는 Mock 로직)의 대대적인 수정이 필요하다.
- **챔피언 숙련도 연동**: Riot API 호출량이 늘어날 수 있고, 현재 로직 복잡도가 증가한다.
- **클립보드 복사**: 구현은 쉽지만, '분석'이라는 핵심 가치보다는 유틸리티 성격이 강해 우선순위가 낮다.

## 3. Spec Draft

### Feature Name
**소환사 최근 20게임 성과 추이 그래프 (Recent Match Performance Trend Graph)**

### Problem Statement
사용자는 최근 자신의 플레이가 개선되고 있는지, 아니면 부진하고 있는지 직관적으로 알기 원한다. 현재의 리스트 뷰는 개별 게임의 디테일을 확인하기엔 좋지만, 전체적인 흐름을 보기엔 부적합하다.

### MVP Scope
- 소환사 상세 페이지 상단(또는 전적 리스트 위)에 꺾은선 그래프(Line Chart) 영역 추가.
- 최근 수집된 20게임(또는 수집된 전체)을 대상으로 함.
- X축: 게임 (최근 순), Y축: KDA Ratio 또는 자체 계산된 퍼포먼스 스코어.
- 각 데이터 포인트(점)에 마우스 오버 시 간략한 정보(챔피언, 킬/데스/어시) 툴팁 표시.

### Optional Scope (Nice to have)
- 승리/패배 구간에 따른 배경색 미세한 차이 또는 포인트 색상 구분 (승리=파랑, 패배=빨강).
- KDA 외에 골드 획득량, 딜량 등으로 지표 전환 기능.

### Acceptance Criteria
1. 소환사 상세 페이지 진입 시, 수집된 매치 데이터가 있다면 그래프가 렌더링되어야 한다.
2. 그래프의 X축은 시간 순서(과거->최근)로 정렬되어야 한다.
3. KDA 계산 시 Death가 0인 경우에 대한 예외 처리(Perfect KDA)가 되어야 한다.
4. 데이터가 0건일 경우 "데이터 부족" 메시지나 빈 상태(Empty State)를 보여줘야 한다.
