# Idea Playbook

## Idea Evaluation Criteria
- Does it improve the core user flow?
- Can it be implemented with current API structure?
- Is it useful even with MockAIProvider?
- Does it strengthen portfolio value?

## Priority Bias
1. User-visible improvements (UI/UX, explanation, insights)
2. Analysis transparency (why this score/recommendation?)
3. DX improvements (API clarity, error messages)
4. Nice-to-have experiments

## Forbidden Ideas
- Paid features
- Social networking features
- Complex ranking systems
- Large-scale infra assumptions
